# RireRoris
​
## サイト概要

### サイトテーマ
### アクセサリー専門サイトRireRoris【RR】
流行りの韓国アクセサリーを低額で購入でき検索をすると自分好みのアクセサリーを探すことのできる通販サイト

​
### テーマを選んだ理由
現在低額で韓国アクセサリーを購入できるサイトは少ないと思っています。私自身学生の時オシャレをしたいと思っていました
ですがおこずかいの範囲内での買い物しかできずもっと低額でオシャレがしたいと思っていましたが、低額でオシャレなアイテムは少なく
代表例で言いますとSHEINの様な低額でオシャレな買い物ができるサイトがもっと増えてほしいと思いました。そこで今回私が提案したいサイトは
低額を強みとしたアクセサリーのみに重点をおいた低額韓国アクセサリー専門ECサイトRireRorisです。
韓国に憧れる学生やお金に困る時期の20代前半、お気持ちプレゼントをオシャレかつ低額で済ませたい方など、低額でおしゃれなアクセサリーを求められている
層をターゲットに「低額でおしゃれな自分好みのアクセサリーを」をコンセプトとしたテーマに致しました。


​
### ターゲットユーザ
・おこずかいの範囲内でオシャレをしたい学生

・オシャレはしたいが金銭面での余裕が無い方

・お気持ち程度のプレゼントをオシャレ + 低額で済ませたい方

・デザインのオシャレなアクセサリーを複数所持したい方
​
### 主な利用シーン
・低額でオシャレをしたい時

・アクセサリーに挑戦したくなる学生の時期などの入り口

・友人、知人、恋人へ低額だがオシャレな品のプレゼントを検討されるとき
